# Express Project

This is a simple Express.js project. Follow the instructions below to get started.

## Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v18 or later recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

## Setup

1. **Clone the repository**

```bash
git clone git@github.com:phamgiaphuc/byway-lms-be.git
cd byway-lms-be
```

2. **Install dependencies**

```bash
npm install
# or
yarn install
```

3. **Environment variables**

```bash
PORT=
DATABASE_URL=
RESEND_API_KEY=
JWT_SECRET=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
```

4. **Running the server**

- Development:

```bash
npm run dev
# or
yarn dev
```

- Production:

```bash
npm start
# or
yarn start
```
